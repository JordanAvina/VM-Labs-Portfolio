Lab 01 - Windows 11 Virtual Machine Setup
=========================================

VM Name: WindowsLab01
Hypervisor: Hyper-V (Windows 10 Pro)
Date: 11/20/2025

---------------------------
1. Objective
---------------------------
- Create a Windows 11 virtual machine using Hyper-V.
- Configure virtual hardware (TPM, Secure Boot, CPU, RAM).
- Install Windows 11 and verify network/drive configuration.
- Document the lab with screenshots for portfolio use.

---------------------------
2. VM Configuration Summary
---------------------------
- Generation: 2
- Startup RAM: 4096 MB (Dynamic Memory enabled)
- Virtual CPUs: 4
- Virtual Disk: 40GB VHDX (dynamic)
- Virtual Switch: External (bridged to physical NIC)
- Secure Boot: Enabled
- TPM: Enabled (required for Windows 11)
- ISO Used: Windows 11 64-bit

---------------------------
3. Steps Completed
---------------------------
1. Created new VM named "WindowsLab01" in Hyper-V Manager.
2. Configured Generation 2 VM and set 4GB RAM with Dynamic Memory.
3. Added 4 virtual processors.
4. Created a 40GB dynamic VHDX.
5. Attached Windows 11 ISO to a virtual DVD drive.
6. Enabled Secure Boot and TPM in Security settings.
7. Adjusted firmware boot order so DVD boots first.
8. Started VM and installed Windows 11 Pro.
9. Completed basic setup and reached the desktop.
10. Verified system information with "winver".
11. Verified network configuration using "ipconfig".
12. Verified disk layout in Disk Management.

---------------------------
4. What I Learned
---------------------------
- How to create and configure a Generation 2 VM.
- Why Secure Boot and TPM are required for Windows 11.
- How Hyper-V handles dynamic memory and dynamic disks.
- How to attach ISO files and manage firmware boot order.
- Basics of disk management inside a VM.
- Confirming network connectivity and addressing with ipconfig.
